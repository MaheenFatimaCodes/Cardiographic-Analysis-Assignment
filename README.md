# 🫀 Cardiotocographic Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Cardiotocography (CTG) dataset** using Python. The goal is to understand the characteristics of fetal health data, identify patterns, detect outliers, and explore relationships between different clinical features through statistical analysis and visualizations.

---

## 🎯 Objectives

* Clean and prepare the dataset for analysis.
* Perform descriptive statistical analysis.
* Explore feature distributions using visualizations.
* Identify outliers and correlations among variables.
* Generate meaningful insights from the data.

---

## 📂 Dataset

The dataset contains cardiotocography measurements used to assess fetal health. It includes various numerical features representing fetal heart rate patterns and other clinical indicators.

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Pandas

* Matplotlib
* Seaborn

## 📊 Exploratory Data Analysis

The analysis includes:

* Data loading and inspection
* Missing value checking
* Duplicate value checking
* Statistical summary
* Univariate analysis
* Bivariate analysis
* Correlation heatmap
* Histograms
* Box plots
* Scatter plots
* Pair plots
* Count plots

---

## 📈 Key Insights

* Explored the distribution of numerical features.
* Identified correlations between clinical measurements.
* Detected potential outliers using box plots.
* Compared feature relationships through scatter plots and pair plots.
* Summarized important trends observed in the dataset.
